# Scheme

### Prefix notation

```scheme
;;; Combination
(+ 5 64)

;;; Fundamental forms
define
lambda
set!
```

### Print

```scheme
(write-line "hello world")
```

### Variables

```scheme
(define pi 3.14159)

(define string "my string")

;;; Local state variables
(set! <name> <new-value>)
```

### Data types

```scheme
;;; List
(list 1 2 3 4)
```

### Procedure

```scheme
(define (square x) (* x x))

;;; Lambda
(lambda (x) (+ x 4))
```
